# 정보

## `//ping` 핑

봇의 현재 응답 속도를 알려줍니다.

```text
//ping

# 별칭
//핑
```

## `//userinfo` 유저 정보

명령어를 입력한 사용자의 정보를 보여줍니다.

```text
//userinfo

# 별칭
//유저정보
//dbwjwjdqh
//ㅕㄴㄷ갸ㅜ래
```

## `//servrinfo` 서버 정보

명령어를 입력한 서버의 정보를 보여줍니다.

```text
//serverinfo

# 별칭
//서버정보
//ㄴㄷㄱㅍㄷ갸ㅜ래
//tjqjwjdqh
```
