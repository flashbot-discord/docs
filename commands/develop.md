# 개발용

**이 명령어는 대부분 봇 소유자로 등록된 사람만 사용할 수 있습니다.**

## `//args-info` 명령어 인수 테스트

봇에서 감지하는 명령어 인수를 알려줍니다.

* 별칭: `argsinfo`, `ㅁㄱㅎㄴ-ㅑㅜ래`, `ㅁㄱㅎ냐ㅜ래`

```text
//args-info <인수...>
```

## ~~`//blacklist` 블랙리스트~~

{% hint style="danger" %} 이 명령어는 아직 완성되지 않았고 작동하지 않습니다! {% endhint %}

이 봇의 블랙리스트(봇을 이용할 수 없는 사람)를 관리합니다.
